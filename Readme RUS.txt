# Mib2 Text Customize Tool

Эта программа предназначена для редактирования файлов .res, содержащих текст в формате Unicode (UTF-16LE). Программа позволяет искать конкретный текст в файле и заменять его другим. Кроме того, вы можете отслеживать историю изменений и сохранять её в виде текстового файла.

## Функции программы:
1. Загрузка и редактирование файлов .res.
2. Поиск конкретных текстовых строк и их замена.
3. Отображение истории изменений (смещение, исходный текст, новый текст).
4. Автоматическая перезагрузка изменённого файла после каждой правки.
5. Возможность сохранения истории изменений в текстовый файл.

## Горячие клавиши:
- **Enter**: Подтвердить замену текста.
- **Ctrl+H**: Показать окно истории изменений.
- **Ctrl+S**: Сохранить историю изменений в текстовый файл.

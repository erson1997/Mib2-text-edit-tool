# Mib2 Text Customize Tool

This program is designed to edit .res files that contain texts in Unicode (UTF-16LE) format. The tool allows you to search for specific text in the file and replace it with another text. Additionally, you can track the history of changes and save it as a text file.

## Program features:
1. Load and edit .res files.
2. Search for specific text strings and replace them.
3. Display the history of changes (offset, original text, new text).
4. Automatically reload the modified file after each change.
5. Save the change history to a text file.

## Keyboard shortcuts:
- **Enter**: Confirm text replacement.
- **Ctrl+H**: Show the change history window.
- **Ctrl+S**: Save the change history to a text file.
